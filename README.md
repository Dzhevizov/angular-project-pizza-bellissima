# Pizza Bellissima

Уеб приложение за онлайн поръчка на пица и италианска кухня, изградено с Angular 17. Клиентите разглеждат менюто, добавят продукти в количката и правят поръчки. Администраторите управляват продуктите и следят дневните поръчки в реално време.

---

## Инсталиране и стартиране

### Изисквания

- [Node.js](https://nodejs.org/) v18 или по-нова версия
- [MongoDB](https://www.mongodb.com/) — работещ локален инстанс на `mongodb://localhost:27017`
- Angular CLI: `npm install -g @angular/cli`

---

### 1. Клониране на репозиторито

```bash
git clone <URL на репозиторито>
cd pizza-bellissima
```

---

### 2. Стартиране на сървъра (backend)

Сървърът е Node.js приложение, намиращо се в папка `server/`.

```bash
cd server
npm install
npm start
```

Сървърът се стартира на **http://localhost:3030**

> Admin панел за преглед на базата данни: **http://localhost:3030/admin**

---

### 3. Стартиране на клиента (Angular)

В нова терминална сесия, от коренната директория на проекта:

```bash
npm install
ng serve
```

Приложението се отваря на **http://localhost:4200**

## Функционалност

### Публична част (достъпна без регистрация)

- **Начална страница** — hero секция, топ оферти с намаления, информация за ресторанта
- **Каталог** — преглед на всички продукти с филтриране по категория (Пици, Паста, Ризото, Десерти, Напитки)
- **Страница с детайли** — пълно описание на продукт с цена в EUR и BGN, бутон за добавяне в количката
- **Регистрация** — създаване на акаунт с лична информация и адрес за доставка
- **Вход** — автентикация с имейл и парола

### Частна част (само за логнати потребители)

- **Количка** — плъзгащ се панел с добавени продукти, автоматично изчисляване на субтотал, отстъпки, такса доставка (безплатна при поръчка над 10 €) и крайна сума в EUR и BGN
- **Поръчване** — потвърждаване на поръчката директно от количката
- **Моите поръчки** — история с всички направени поръчки, статус (Изчакваща / В доставка / Завършена / Отказана), автоматично опресняване на всеки 30 секунди

### Администраторска част

- **Добавяне на продукт** — форма за нов продукт с превю на изображение и конвертиране на цената EUR → BGN
- **Редактиране на продукт** — обновяване на данните за съществуващ продукт
- **Изтриване на продукт** — премахване от каталога
- **Дневни поръчки** — преглед на всички поръчки за деня с пълни детайли и суми

## Структура на проекта

```
pizza-bellissima/        # Angular клиент
├── src/app/
│   ├── components/      # UI компоненти (home, catalog, cart, orders и др.)
│   ├── guards/          # authGuard, adminGuard, guestGuard
│   ├── interceptors/    # HTTP auth interceptor (X-Authorization header)
│   ├── models/          # TypeScript интерфейси (User, Product, Order)
│   ├── pipes/           # eurToLev pipe (EUR → BGN конвертиране)
│   └── services/        # AuthService, ProductService, CartService, OrderService
└── ...

server/                  # Node.js backend
├── server.js            # REST API + MongoDB (порт 3030)
└── package.json
```

## Роли

| Роля | Достъп |
|------|--------|
| Гост | Начална страница, каталог, детайли, вход, регистрация |
| Клиент | + Количка, поръчване, история на поръчките |
| Администратор | + Добавяне / редактиране / изтриване на продукти, дневни поръчки |

> Администраторски акаунт се създава чрез ръчно задаване на `"role": "admin"` в MongoDB колекцията `users`.
